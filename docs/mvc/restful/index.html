RESTful风格API

RESTful是一种规范和风格，不是一种技术，如果你的API满足这种风格那就是RESTful风格的API

RESTful成熟度

  • 层级1: 使用HTTP作为网络传输协议
  • 层级2: URL表示资源
  • 层级3: 使用HTTP方法表示行为方式
  • 层级4: 超媒体控制
